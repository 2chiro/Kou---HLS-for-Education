# Kou - High-level Synthesis for Education

## Install

First, clone the repo the new project folder.

```
git clone https://github.com/2chiro/Kou-HLS-for-Education.git project-name
```

And then install the dependencies with npm.

```
cd project-name
```

## Usage

### Build the app

```
npm run build
```

### Run the app

```
npm start
```
